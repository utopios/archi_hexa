Architecture Hexagonale

Ports & Adapters, DDD, Clean Architecture, Tests & CQRS

3 jours (21h) Intermédiaire - Avancé Java 17+ / Spring Boot 3.x 6-12 stagiaires
Objectif de la formation : Maîtriser l'architecture hexagonale pour concevoir des applications Java maintenables, testables et évolutives. Du domaine métier aux adapteurs d'infrastructure, en passant par la pyramide des tests et la construction itérative BDD/TDD.
